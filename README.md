# ShopChat 🛍️
A chat-enabled online shopping system with real-time buyer-seller communication.

## Tech Stack
MongoDB · Express.js · React.js · Node.js · Firebase · Socket.io

## Features
- Role-based authentication (Buyer / Seller) via Firebase
- Product listing, browsing, and search
- Add to cart and order placement
- Real-time one-to-one chat using WebSockets (Socket.io)
- Online/offline user status tracking
- Real-time notifications for new messages and orders
- Message history storage and retrieval
